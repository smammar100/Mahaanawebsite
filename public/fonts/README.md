# Switzer font files (optional – for body copy)

Body copy is set to **Switzer** in `src/app/globals.css` via `@font-face`. The build succeeds without these files; the browser then falls back to system font until you add them.

To enable Switzer, download from [fontshare.com/fonts/switzer](https://www.fontshare.com/fonts/switzer) (free) and place the **woff2** files in this directory with these **exact** names:

- Switzer-Light.woff2 (weight 300)
- Switzer-Regular.woff2 (400)
- Switzer-Medium.woff2 (500)
- Switzer-Semibold.woff2 (600)
- Switzer-Bold.woff2 (700)
- Switzer-Extrabold.woff2 (800)

Headings use Outfit (Google Font).
