# Switzer font files

Download Switzer from [fontshare.com/fonts/switzer](https://www.fontshare.com/fonts/switzer) and place the woff2 files here:

- Switzer-Light.woff2 (300)
- Switzer-Regular.woff2 (400)
- Switzer-Medium.woff2 (500)
- Switzer-Semibold.woff2 (600)
- Switzer-Bold.woff2 (700)
- Switzer-Extrabold.woff2 (800)

If your downloaded files use different names, rename them to match or update the @font-face paths in globals.css.
